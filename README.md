# Fashion Aura

Fashion Aura is a frontend-only fashion e-commerce website built with React, TypeScript, Tailwind CSS, and Vite.

## Run locally

1. Install Node.js 20 or newer.
2. Install dependencies with `npm install`.
3. Start the development server with `npm run dev`.

## Validate a production build

- Run `npm run lint` to check TypeScript.
- Run `npm run build` to create the production files in `dist`.
